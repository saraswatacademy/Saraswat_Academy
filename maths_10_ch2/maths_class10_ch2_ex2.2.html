Exercise 2.2 – Question 1

Find the zeroes of the following quadratic polynomials and verify the relationship between the zeroes and the coefficients.

For a quadratic polynomial ax2 + bx + c:

Sum of zeroes = −b/a

Product of zeroes = c/a

(i) x2 − 2x − 8

x2 − 2x − 8 = (x − 4)(x + 2)

Zeroes = 4, −2

Sum of zeroes = 4 + (−2) = 2

−b/a = −(−2)/1 = 2 ✓

Product of zeroes = 4 × (−2) = −8

c/a = −8/1 = −8 ✓

(ii) 4s2 − 4s + 1

4s2 − 4s + 1 = (2s − 1)2

Zeroes = 1/2, 1/2

Sum of zeroes = 1/2 + 1/2 = 1

−b/a = −(−4)/4 = 1 ✓

Product of zeroes = (1/2)(1/2) = 1/4

c/a = 1/4 ✓

(iii) 6x2 − 3 − 7x

Rearrange:

6x2 − 7x − 3

= 6x2 − 9x + 2x − 3

= 3x(2x − 3) + 1(2x − 3)

= (3x + 1)(2x − 3)

Zeroes = −1/3, 3/2

Sum of zeroes = −1/3 + 3/2

= (−2 + 9)/6

= 7/6

−b/a = −(−7)/6 = 7/6 ✓

Product of zeroes = (−1/3)(3/2) = −1/2

c/a = −3/6 = −1/2 ✓

(iv) 4u2 + 8u

4u(u + 2) = 0

Zeroes = 0, −2

Sum of zeroes = 0 + (−2) = −2

−b/a = −8/4 = −2 ✓

Product of zeroes = 0 × (−2) = 0

c/a = 0/4 = 0 ✓

(v) t2 − 15

t2 − 15 = 0

t = ±√15

Zeroes = √15, −√15

Sum of zeroes = √15 − √15 = 0

−b/a = 0 ✓

Product of zeroes = (√15)(−√15)

= −15

c/a = −15 ✓

(vi) 3x2 − x − 4

3x2 − x − 4

= 3x2 − 4x + 3x − 4

= x(3x − 4) + 1(3x − 4)

= (x + 1)(3x − 4)

Zeroes = −1, 4/3

Sum of zeroes = −1 + 4/3

= 1/3

−b/a = −(−1)/3 = 1/3 ✓

Product of zeroes = (−1)(4/3)

= −4/3

c/a = −4/3 ✓

Question 2

Find a quadratic polynomial whose zeroes are the given numbers.

If the zeroes are α and β, then

Polynomial = x2 − (α + β)x + αβ

(i) Zeroes = 1/4, −1

Sum = 1/4 − 1 = −3/4

Product = −1/4

Polynomial:

x2 + (3/4)x − 1/4

Multiplying by 4:

4x2 + 3x − 1

(ii) Zeroes = √2, 1/3

Sum = √2 + 1/3

Product = √2/3

Polynomial:

x2 − (√2 + 1/3)x + √2/3

Multiplying by 3:

3x2 − (3√2 + 1)x + √2

(iii) Zeroes = 0, √5

Sum = √5

Product = 0

Polynomial:

x2 − √5x

x(x − √5)

(iv) Zeroes = 1, 1

Sum = 2

Product = 1

Polynomial:

x2 − 2x + 1

(x − 1)2

(v) Zeroes = −1/4, 1/4

Sum = 0

Product = −1/16

Polynomial:

x2 − 1/16

Multiplying by 16:

16x2 − 1

(vi) Zeroes = 4, 1

Sum = 5

Product = 4

Polynomial:

x2 − 5x + 4
