EXERCISE 4.3

1. Find the nature of the roots of the following quadratic equations. If the real roots exist, find them:

(i) 2x2 − 3x + 5 = 0

Solution:

Comparing the given equation with the standard form ax2 + bx + c = 0, we get:

a = 2, b = −3, c = 5

The discriminant (D) is given by:

D = b2 − 4ac

D = (−3)2 − 4(2)(5)

D = 9 − 40

D = −31

Since D < 0, the given equation has no real roots.

(ii) 3x2 − 4√3 x + 4 = 0

Solution:

Comparing with the standard form ax2 + bx + c = 0, we get:

a = 3, b = −4√3, c = 4

The discriminant (D) is given by:

D = b2 − 4ac

D = (−4√3)2 − 4(3)(4)

D = 48 − 48

D = 0

Since D = 0, the given equation has two equal real roots.

The roots are given by x = −b / 2a:

x = −(−4√3) / 2(3)

x = 4√3 / 6

x = 2√3 / 3 = 2 / √3

Hence, the roots are 2 / √3 and 2 / √3.

(iii) 2x2 − 6x + 3 = 0

Solution:

Comparing with the standard form ax2 + bx + c = 0, we get:

a = 2, b = −6, c = 3

The discriminant (D) is given by:

D = b2 − 4ac

D = (−6)2 − 4(2)(3)

D = 36 − 24

D = 12

Since D > 0, the given equation has two distinct real roots.

Using the quadratic formula, x = (−b ± √D) / 2a:

x = (−(−6) ± √12) / 2(2)

x = (6 ± 2√3) / 4

Taking out 2 as a common factor:

x = 2(3 ± √3) / 4

x = (3 ± √3) / 2

Hence, the roots are (3 + √3) / 2 and (3 − √3) / 2.

2. Find the values of k for each of the following quadratic equations, so that they have two equal roots.

(i) 2x2 + kx + 3 = 0

Solution:

Comparing with the standard form ax2 + bx + c = 0, we get:

a = 2, b = k, c = 3

For two equal roots, the discriminant must be equal to zero (D = 0):

b2 − 4ac = 0

k2 − 4(2)(3) = 0

k2 − 24 = 0

k2 = 24

k = ±√24

k = ±2√6

Hence, the values of k are 2√6 and −2√6.

(ii) kx(x − 2) + 6 = 0

Solution:

Expanding the given equation to convert it into standard form:

kx2 − 2kx + 6 = 0

Comparing with the standard form ax2 + bx + c = 0, we get:

a = k, b = −2k, c = 6

For two equal roots, the discriminant must be equal to zero (D = 0):

b2 − 4ac = 0

(−2k)2 − 4(k)(6) = 0

4k2 − 24k = 0

4k(k − 6) = 0

Either 4k = 0 or k − 6 = 0

k = 0 or k = 6

If k = 0, the terms involving x2 and x become 0, which means the equation is no longer quadratic. Therefore, we reject k = 0.

Hence, the value of k is 6.

3. Is it possible to design a rectangular mango grove whose length is twice its breadth, and the area is 800 m2? If so, find its length and breadth.

Solution:

Let the breadth of the rectangular mango grove be x metres.

Since the length is twice its breadth, the length will be 2x metres.

The area of the rectangular grove is given by:

Area = Length × Breadth

800 = (2x)(x)

800 = 2x2

x2 = 400

x2 − 400 = 0

Comparing with the standard form ax2 + bx + c = 0, we get a = 1, b = 0, c = −400.

Let us check the discriminant to see if real solutions exist:

D = b2 − 4ac = 02 − 4(1)(−400) = 1600

Since D > 0, real solutions exist. Therefore, it is possible to design the mango grove.

Solving for x:

x = ±√400

x = ±20

Since breadth cannot be negative, we reject x = −20. Thus, x = 20.

Breadth = 20 m

Length = 2(20) = 40 m

Hence, designing the grove is possible, with a breadth of 20 m and a length of 40 m.

4. Is the following situation possible? If so, determine their present ages. The sum of the ages of two friends is 20 years. Four years ago, the product of their ages in years was 48.

Solution:

Let the present age of the first friend be x years.

Since the sum of their ages is 20 years, the present age of the second friend will be (20 − x) years.

Four years ago:

Age of the first friend = (x − 4) years

Age of the second friend = (20 − x − 4) = (16 − x) years

According to the given condition, the product of their ages four years ago was 48:

(x − 4)(16 − x) = 48

16x − x2 − 64 + 4x = 48

−x2 + 20x − 64 − 48 = 0

−x2 + 20x − 112 = 0

Multiplying the entire equation by −1:

x2 − 20x + 112 = 0

Comparing with the standard form ax2 + bx + c = 0, we get a = 1, b = −20, c = 112.

Let us check the discriminant to determine if the situation is possible:

D = b2 − 4ac

D = (−20)2 − 4(1)(112)

D = 400 − 448

D = −48

Since D < 0, the equation has no real roots. Therefore, the given situation is not possible.

5. Is it possible to design a rectangular park of perimeter 80 m and area 400 m2? If so, find its length and breadth.

Solution:

Let the length of the rectangular park be x metres and its breadth be y metres.

The perimeter of a rectangle is given by 2(length + breadth):

2(x + y) = 80

x + y = 40

y = 40 − x

The area of the rectangular park is given by length × breadth:

Area = x × y = 400

Substituting the value of y from the perimeter relation:

x(40 − x) = 400

40x − x2 = 400

Rearranging into standard form:

x2 − 40x + 400 = 0

Comparing with the standard form ax2 + bx + c = 0, we get a = 1, b = −40, c = 400.

Let us check the discriminant to determine if this design is possible:

D = b2 − 4ac

D = (−40)2 − 4(1)(400)

D = 1600 − 1600

D = 0

Since D = 0, real and equal roots exist. Therefore, it is possible to design the rectangular park.

Solving for x using factorisation:

x2 − 20x − 20x + 400 = 0

x(x − 20) − 20(x − 20) = 0

(x − 20)(x − 20) = 0

x = 20

So, Length = 20 m

Breadth = 40 − 20 = 20 m

Hence, designing the park is possible, and it will be a square shape with a length of 20 m and a breadth of 20 m.
