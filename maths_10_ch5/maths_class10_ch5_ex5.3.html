EXERCISE 5.3

Core Formulas Used:

  • nth term of an AP: an = a + (n - 1)d
  • Sum of first n terms: Sn = n⁄2 [2a + (n - 1)d] or Sn = n⁄2 (a + an)

Question 1: Find the sum of the following APs

(i) 2, 7, 12, ..., to 10 terms

Given: First term (a) = 2, Common difference (d) = 7 - 2 = 5, Number of terms (n) = 10.
Using formula: Sn = n⁄2 [2a + (n - 1)d]
S10 = 10⁄2 [2(2) + (10 - 1)5]
S10 = 5 [4 + 9(5)]
S10 = 5 [4 + 45]
S10 = 5 × 49 = 245

(ii) -37, -33, -29, ..., to 12 terms

Given: a = -37, d = -33 - (-37) = 4, n = 12.
S12 = 12⁄2 [2(-37) + (12 - 1)4]
S12 = 6 [-74 + 11(4)]
S12 = 6 [-74 + 44]
S12 = 6 [-30] = -180

(iii) 0.6, 1.7, 2.8, ..., to 100 terms

Given: a = 0.6, d = 1.7 - 0.6 = 1.1, n = 100.
S100 = 100⁄2 [2(0.6) + (100 - 1)1.1]
S100 = 50 [1.2 + 99(1.1)]
S100 = 50 [1.2 + 108.9]
S100 = 50 × 110.1 = 5505

(iv) 1⁄15, 1⁄12, 1⁄10, ..., to 11 terms

Given: a = 1⁄15, d = 1⁄12 - 1⁄15 = (5 - 4)⁄60 = 1⁄60, n = 11.
S11 = 11⁄2 [2(1⁄15) + (11 - 1)(1⁄60)]
S11 = 11⁄2 [2⁄15 + 10⁄60]
S11 = 11⁄2 [2⁄15 + 1⁄6] = 11⁄2 [(4 + 5)⁄30]
S11 = 11⁄2 [9⁄30] = 11⁄2 [3⁄10] = 33⁄20

Question 2: Find the sums given below

(i) 7 + 101⁄2 + 14 + ... + 84

Given: a = 7, d = 10.5 - 7 = 3.5 = 7⁄2, Last term (an) = 84.
First, find n: an = a + (n - 1)d
84 = 7 + (n - 1)(7⁄2)
77 = (n - 1)(7⁄2)
11 = (n - 1)⁄2 ⇒ n - 1 = 22 ⇒ n = 23.
Now, find sum: S23 = 23⁄2 (a + an)
S23 = 23⁄2 (7 + 84) = 23⁄2 (91) = 2093⁄2 = 10461⁄2

(ii) 34 + 32 + 30 + ... + 10

Given: a = 34, d = 32 - 34 = -2, an = 10.
Find n: 10 = 34 + (n - 1)(-2)
-24 = (n - 1)(-2) ⇒ n - 1 = 12 ⇒ n = 13.
Find sum: S13 = 13⁄2 (34 + 10)
S13 = 13⁄2 (44) = 13 × 22 = 286

(iii) -5 + (-8) + (-11) + ... + (-230)

Given: a = -5, d = -8 - (-5) = -3, an = -230.
Find n: -230 = -5 + (n - 1)(-3)
-225 = (n - 1)(-3) ⇒ n - 1 = 75 ⇒ n = 76.
Find sum: S76 = 76⁄2 [-5 + (-230)]
S76 = 38 × (-235) = -8930

Question 3: In an AP

(i) Given a = 5, d = 3, an = 50, find n and Sn

50 = 5 + (n - 1)3 ⇒ 45 = (n - 1)3 ⇒ n - 1 = 15 ⇒ n = 16
S16 = 16⁄2 (5 + 50) = 8 × 55 = 440

(ii) Given a = 7, a13 = 35, find d and S13

a13 = a + 12d ⇒ 35 = 7 + 12d ⇒ 28 = 12d ⇒ d = 7⁄3
S13 = 13⁄2 (7 + 35) = 13⁄2 (42) = 13 × 21 = 273

(iii) Given a12 = 37, d = 3, find a and S12

a12 = a + 11d ⇒ 37 = a + 11(3) ⇒ 37 = a + 33 ⇒ a = 4
S12 = 12⁄2 (4 + 37) = 6 × 41 = 246

(iv) Given a3 = 15, S10 = 125, find d and a10

From a3: a + 2d = 15 — (Eq 1)
From S10: 10⁄2 [2a + 9d] = 125 ⇒ 5 [2a + 9d] = 125 ⇒ 2a + 9d = 25 — (Eq 2)
Multiply Eq 1 by 2: 2a + 4d = 30
Subtracting this from Eq 2: 5d = -5 ⇒ d = -1
Substitute d in Eq 1: a + 2(-1) = 15 ⇒ a = 17.
a10 = a + 9d = 17 + 9(-1) = 8

(v) Given d = 5, S9 = 75, find a and a9

S9 = 9⁄2 [2a + 8(5)] = 75
9⁄2 (2a + 40) = 75 ⇒ 9(a + 20) = 75 ⇒ 9a + 180 = 75 ⇒ 9a = -105 ⇒ a = -35⁄3
a9 = a + 8d = -35⁄3 + 8(5) = -35⁄3 + 40 = (-35 + 120)⁄3 = 85⁄3

(vi) Given a = 2, d = 8, Sn = 90, find n and an

90 = n⁄2 [2(2) + (n - 1)8] ⇒ 180 = n [4 + 8n - 8] ⇒ 180 = n (8n - 4)
180 = 8n2 - 4n ⇒ 2n2 - n - 45 = 0
Solving the quadratic: 2n2 - 10n + 9n - 45 = 0 ⇒ 2n(n - 5) + 9(n - 5) = 0
(2n + 9)(n - 5) = 0 ⇒ Since n cannot be negative/fractional, n = 5
a5 = a + 4d = 2 + 4(8) = 34

(vii) Given a = 8, an = 62, Sn = 210, find n and d

Sn = n⁄2 (a + an) ⇒ 210 = n⁄2 (8 + 62)
210 = n⁄2 × 70 ⇒ 210 = 35n ⇒ n = 6
a6 = a + 5d ⇒ 62 = 8 + 5d ⇒ 54 = 5d ⇒ d = 54⁄5

(viii) Given an = 4, d = 2, Sn = -14, find n and a

an = a + (n - 1)d ⇒ 4 = a + (n - 1)2 ⇒ a = 4 - 2n + 2 ⇒ a = 6 - 2n — (Eq 1)
Sn = n⁄2 (a + an) ⇒ -14 = n⁄2 (6 - 2n + 4)
-28 = n (10 - 2n) ⇒ -28 = 10n - 2n2 ⇒ 2n2 - 10n - 28 = 0
n2 - 5n - 14 = 0 ⇒ (n - 7)(n + 2) = 0 ⇒ n = 7 (since n must be positive)
Substitute n into Eq 1: a = 6 - 2(7) = -8

(ix) Given a = 3, n = 8, S = 192, find d

192 = 8⁄2 [2(3) + (8 - 1)d]
192 = 4 [6 + 7d] ⇒ 48 = 6 + 7d ⇒ 42 = 7d ⇒ d = 6

(x) Given l = 28, S = 144, and there are total 9 terms. Find a

Note: l is the last term (a9 = 28).
S9 = 9⁄2 (a + l)
144 = 9⁄2 (a + 28)
(144 × 2)⁄9 = a + 28 ⇒ 16 × 2 = a + 28 ⇒ 32 = a + 28 ⇒ a = 4

Question 4: How many terms of the AP: 9, 17, 25, . . . must be taken to give a sum of 636?

Given: First term (a) = 9, Common difference (d) = 17 - 9 = 8, Sum (Sn) = 636.
Using formula: Sn = n⁄2 [2a + (n - 1)d]
636 = n⁄2 [2(9) + (n - 1)8]
636 = n⁄2 [18 + 8n - 8]
636 = n⁄2 [10 + 8n]
636 = n(5 + 4n)
636 = 5n + 4n2 ⇒ 4n2 + 5n - 636 = 0
Solving the quadratic equation using the formula n = (-b ± √(b2 - 4ac))⁄2a:
n = (-5 ± √(52 - 4(4)(-636)))⁄2(4)
n = (-5 ± √(25 + 10176))⁄8
n = (-5 ± √10201)⁄8 = (-5 ± 101)⁄8
Taking the positive value: n = 96⁄8 = 12
Thus, 12 terms must be taken.

Question 5: The first term of an AP is 5, the last term is 45 and the sum is 400. Find the number of terms and the common difference.

Given: a = 5, Last term (l or an) = 45, Sn = 400.
Using formula: Sn = n⁄2 (a + l)
400 = n⁄2 (5 + 45)
400 = n⁄2 (50) ⇒ 400 = 25n ⇒ n = 16
Now, using formula: an = a + (n - 1)d
45 = 5 + (16 - 1)d
40 = 15d ⇒ d = 40⁄15 = 8⁄3

Question 6: The first and the last terms of an AP are 17 and 350 respectively. If the common difference is 9, how many terms are there and what is their sum?

Given: a = 17, l = 350, d = 9.
Using formula: an = a + (n - 1)d
350 = 17 + (n - 1)9
333 = (n - 1)9 ⇒ n - 1 = 37 ⇒ n = 38
Now, find the sum: S38 = 38⁄2 (a + l)
S38 = 19 × (17 + 350) = 19 × 367 = 6973

Question 7: Find the sum of first 22 terms of an AP in which d = 7 and 22nd term is 149.

Given: n = 22, d = 7, a22 = 149.
Using formula: a22 = a + 21d
149 = a + 21(7)
149 = a + 147 ⇒ a = 2
Now, find the sum: S22 = 22⁄2 (a + a22)
S22 = 11 × (2 + 149) = 11 × 151 = 1661

Question 8: Find the sum of first 51 terms of an AP whose second and third terms are 14 and 18 respectively.

Given: a2 = 14, a3 = 18, n = 51.
Common difference (d) = a3 - a2 = 18 - 14 = 4
First term (a) = a2 - d = 14 - 4 = 10
Now, find the sum: S51 = 51⁄2 [2a + (n - 1)d]
S51 = 51⁄2 [2(10) + (51 - 1)4]
S51 = 51⁄2 [20 + 50(4)] = 51⁄2 [220]
S51 = 51 × 110 = 5610

Question 9: If the sum of first 7 terms of an AP is 49 and that of 17 terms is 289, find the sum of first n terms.

Given: S7 = 49, S17 = 289.
From S7: 7⁄2 [2a + 6d] = 49 ⇒ 7(a + 3d) = 49 ⇒ a + 3d = 7 — (Eq 1)
From S17: 17⁄2 [2a + 16d] = 289 ⇒ 17(a + 8d) = 289 ⇒ a + 8d = 17 — (Eq 2)
Subtracting Eq 1 from Eq 2: 5d = 10 ⇒ d = 2
Substitute d = 2 in Eq 1: a + 3(2) = 7 ⇒ a + 6 = 7 ⇒ a = 1
Now, find Sn: Sn = n⁄2 [2(1) + (n - 1)2]
Sn = n⁄2 [2 + 2n - 2] = n⁄2 (2n) = n2

Question 10: Show that a1, a2, . . . , an, . . . form an AP where an is defined as below:

(i) an = 3 + 4n. Also find the sum of the first 15 terms.

Let's find the first few terms:
a1 = 3 + 4(1) = 7
a2 = 3 + 4(2) = 11
a3 = 3 + 4(3) = 15
Since the difference between consecutive terms (11 - 7 = 4, 15 - 11 = 4) is constant, it forms an AP with a = 7 and d = 4.
Sum of first 15 terms: S15 = 15⁄2 [2(7) + (15 - 1)4]
S15 = 15⁄2 [14 + 14(4)] = 15⁄2 [14 + 56] = 15⁄2 (70)
S15 = 15 × 35 = 525

(ii) an = 9 - 5n. Also find the sum of the first 15 terms.

Let's find the first few terms:
a1 = 9 - 5(1) = 4
a2 = 9 - 5(2) = -1
a3 = 9 - 5(3) = -6
Since the difference between consecutive terms (-1 - 4 = -5, -6 - (-1) = -5) is constant, it forms an AP with a = 4 and d = -5.
Sum of first 15 terms: S15 = 15⁄2 [2(4) + (15 - 1)(-5)]
S15 = 15⁄2 [8 + 14(-5)] = 15⁄2 [8 - 70] = 15⁄2 (-62)
S15 = 15 × (-31) = -465

Question 11: If the sum of the first n terms of an AP is 4n - n2, what is the first term (that is S1)? What is the sum of first two terms? What is the second term? Similarly, find the 3rd, the 10th and the nth terms.

Given: Sn = 4n - n2
First term (a1) = S1 = 4(1) - (1)2 = 4 - 1 = 3
Sum of first two terms (S2) = 4(2) - (2)2 = 8 - 4 = 4
Second term (a2) = S2 - S1 = 4 - 3 = 1
Common difference (d) = a2 - a1 = 1 - 3 = -2

Third term (a3) = a + 2d = 3 + 2(-2) = 3 - 4 = -1
Tenth term (a10) = a + 9d = 3 + 9(-2) = 3 - 18 = -15
nth term (an) = a + (n - 1)d = 3 + (n - 1)(-2) = 3 - 2n + 2 = 5 - 2n

Question 12: Find the sum of the first 40 positive integers divisible by 6.

The sequence is: 6, 12, 18, 24, . . . to 40 terms.
Given: a = 6, d = 6, n = 40.
S40 = 40⁄2 [2(6) + (40 - 1)6]
S40 = 20 [12 + 39(6)] = 20 [12 + 234]
S40 = 20 × 246 = 4920

Question 13: Find the sum of the first 15 multiples of 8.

The sequence is: 8, 16, 24, 32, . . . to 15 terms.
Given: a = 8, d = 8, n = 15.
S15 = 15⁄2 [2(8) + (15 - 1)8]
S15 = 15⁄2 [16 + 14(8)] = 15⁄2 [16 + 112]
S15 = 15⁄2 (128) = 15 × 64 = 960

Question 14: Find the sum of the odd numbers between 0 and 50.

The odd numbers between 0 and 50 are: 1, 3, 5, 7, . . . , 49.
This is an AP where a = 1, d = 2, and last term (l) = 49.
First find n: 49 = 1 + (n - 1)2 ⇒ 48 = (n - 1)2 ⇒ n - 1 = 24 ⇒ n = 25.
Now find the sum: S25 = 25⁄2 (a + l)
S25 = 25⁄2 (1 + 49) = 25⁄2 (50) = 25 × 25 = 625

Question 15: A contract on construction job specifies a penalty for delay of completion beyond a certain date as follows: ₹ 200 for the first day, ₹ 250 for the second day, ₹ 300 for the third day, etc., the penalty for each succeeding day being ₹ 50 more than for the preceding day. How much money the contractor has to pay as penalty, if he has delayed the work by 30 days?

Given:
Penalty for the first day (a) = ₹ 200
Penalty for the second day = ₹ 250
Penalty for the third day = ₹ 300
Since the penalty increases by a fixed amount daily, it forms an AP with a common difference (d) = ₹ 50.
Number of days delayed (n) = 30.

Using the sum formula: Sn = n⁄2 [2a + (n - 1)d]
S30 = 30⁄2 [2(200) + (30 - 1)50]
S30 = 15 [400 + 29 × 50]
S30 = 15 [400 + 1450]
S30 = 15 [1850]
S30 = ₹ 27,750

Thus, the contractor has to pay ₹ 27,750 as a penalty.

Question 16: A sum of ₹ 700 is to be used to give seven cash prizes to students of a school for their overall academic performance. If each prize is ₹ 20 less than its preceding prize, find the value of each of the prizes.

Given:
Total budget for prizes (Sn) = ₹ 700
Total number of prizes (n) = 7
Since each prize is ₹ 20 less than the preceding one, the common difference (d) = -20.

Using the sum formula: Sn = n⁄2 [2a + (n - 1)d]
700 = 7⁄2 [2a + (7 - 1)(-20)]
(700 × 2)⁄7 = 2a + 6(-20)
200 = 2a - 120
320 = 2a ⇒ a = ₹ 160

Now, calculating the value of each of the 7 prizes:
1st prize = a = ₹ 160
2nd prize = a - 20 = ₹ 140
3rd prize = 140 - 20 = ₹ 120
4th prize = 120 - 20 = ₹ 100
5th prize = 100 - 20 = ₹ 80
6th prize = 80 - 20 = ₹ 60
7th prize = 60 - 20 = ₹ 40

Question 17: In a school, students thought of planting trees in and around the school to reduce air pollution. It was decided that the number of trees, that each section of each class will plant, will be the same as the class, in which they are studying, e.g., a section of Class I will plant 1 tree, a section of Class II will plant 2 trees and so on till Class XII. There are three sections of each class. How many trees will be planted by the students?

Given:
Number of classes = 12 (from Class I to Class XII)
Number of sections per class = 3

Let's first find the number of trees planted by a single section of all classes from I to XII:
The sequence is: 1, 2, 3, 4, . . . , 12.
This is an AP where first term (a) = 1, last term (l) = 12, and number of terms (n) = 12.

Sum of trees for 1 section: Sn = n⁄2 (a + l)
S12 = 12⁄2 (1 + 12)
S12 = 6 × 13 = 78 trees

Since there are 3 sections in each class, the total number of trees planted:
Total trees = 3 × (Trees planted by 1 section)
Total trees = 3 × 78 = 234 trees

Thus, a total of 234 trees will be planted by the students.

Question 18: A spiral is made up of successive semicircles, with centres alternately at A and B, starting with centre at A, of radii 0.5 cm, 1.0 cm, 1.5 cm, 2.0 cm, . . . What is the total length of such a spiral made up of thirteen consecutive semicircles? (Take π = 22⁄7)



Derivation and Logic:
The perimeter (length) of a semicircle with radius r is given by the formula: l = πr

Let the lengths of the successive semicircles be l1, l2, l3, . . . , l13.
l1 = π × 0.5 cm
l2 = π × 1.0 cm
l3 = π × 1.5 cm

The total length of the spiral (L) is the sum of these thirteen lengths:
L = l1 + l2 + l3 + . . . + l13
L = π(0.5) + π(1.0) + π(1.5) + . . . + π(r13)
L = π [0.5 + 1.0 + 1.5 + . . . \text{ up to 13 terms}]

The sequence inside the bracket forms an Arithmetic Progression (AP):
First term (a) = 0.5
Common difference (d) = 1.0 - 0.5 = 0.5
Number of terms (n) = 13

Using the sum formula for an AP: Sn = n⁄2 [2a + (n - 1)d]
S13 = 13⁄2 [2(0.5) + (13 - 1)0.5]
S13 = 13⁄2 [1.0 + 12(0.5)]
S13 = 13⁄2 [1.0 + 6.0]
S13 = 13⁄2 [7] = 91⁄2

Now substitute S13 back into the total length equation along with π = 22⁄7:
L = π × S13
L = 22⁄7 × 91⁄2
L = (22⁄2) × (91⁄7)
L = 11 × 13 = 143 cm

Thus, the total length of the spiral made up of thirteen consecutive semicircles is 143 cm.

Question 19: 200 logs are stacked in the following manner: 20 logs in the bottom row, 19 in the next row, 18 in the row next to it and so on. In how many rows are the 200 logs placed and how many logs are in the top row?



Given:
Logs in the bottom row (1st row), a = 20
Logs in the next row (2nd row) = 19
Logs in the row next to it (3rd row) = 18

The number of logs in consecutive rows forms an Arithmetic Progression (AP): 20, 19, 18, . . .
First term (a) = 20
Common difference (d) = 19 - 20 = -1
Total number of logs (Sn) = 200

Using the sum formula for an AP: Sn = n⁄2 [2a + (n - 1)d]
200 = n⁄2 [2(20) + (n - 1)(-1)]
400 = n [40 - n + 1]
400 = n [41 - n]
400 = 41n - n2

Rearranging this into a standard quadratic equation form:
n2 - 41n + 400 = 0

Solving by splitting the middle term:
n2 - 16n - 25n + 400 = 0
n(n - 16) - 25(n - 16) = 0
(n - 16)(n - 25) = 0

This gives two potential values for the number of rows:
n = 16 or n = 25

Verification of terms:
Let's determine which value is physically possible by finding the number of logs in the top row (an):

Case 1: If n = 25
a25 = a + 24d = 20 + 24(-1) = 20 - 24 = -4
Since the number of logs in a row cannot be negative, n = 25 is rejected.

Case 2: If n = 16
a16 = a + 15d = 20 + 15(-1) = 20 - 15 = 5
Since this yields a positive count of logs, n = 16 is accepted.

Therefore, the 200 logs are placed in 16 rows, and there are 5 logs in the top row.

Question 20: In a potato race, a bucket is placed at the starting point, which is 5 m from the first potato, and the other potatoes are placed 3 m apart in a straight line. There are ten potatoes in the line (see Fig. 5.6). A competitor starts from the bucket, picks up the nearest potato, runs back with it, drops it in the bucket, runs back to pick up the next potato, runs to the bucket to drop it in, and she continues in the same way until all the potatoes are in the bucket. What is the total distance the competitor has to run?



Given:
Distance of the first potato from the bucket = 5 m
Distance between subsequent potatoes = 3 m
Total number of potatoes (n) = 10

Distance analysis:
To pick up and drop each potato, the competitor must run to the potato and return to the bucket. Therefore, the distance run for each potato is double the distance from the bucket.

Distance run for the 1st potato = 2 × 5 m = 10 m
Distance run for the 2nd potato = 2 × (5 + 3) m = 2 × 8 m = 16 m
Distance run for the 3rd potato = 2 × (5 + 3 + 3) m = 2 × 11 m = 22 m

The distances run for successive potatoes form an Arithmetic Progression (AP): 10, 16, 22, . . .
First term (a) = 10
Common difference (d) = 16 - 10 = 6
Number of terms (n) = 10

Using the sum formula for an AP: Sn = n⁄2 [2a + (n - 1)d]
S10 = 10⁄2 [2(10) + (10 - 1)6]
S10 = 5 [20 + 9 × 6]
S10 = 5 [20 + 54]
S10 = 5 × 74
S10 = 370 m

Thus, the total distance the competitor has to run is 370 meters.
